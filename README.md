Deskripsi database :
Database berisi 5 tabel yang membentuk satu sistem bernama Kasir Sederhana. Tabel-tabel tersebut antara lain Tabel Pegawai, Tabel barang, tabel login, tabel tranasaksi dan tabel member.

Penjelasan Tabel :
Tabel Pegawai berisi id_pegawai (primary key), nama, jenis kelamin,alamat dan no hp
Tabel barang berisi id_barang, nama, dan jumlah
Tabel login berisi id_login, username dan password
Tabel Transaksi berisi id_transaksi, nama, jumlah, total harga dan total bayar
Tabel Member berisi id_member, nama, alamat dan no_hp

Cara import database :
jika ingin melihat tampilan/import database di phpmyadmin anda hanya perlu beberapa langkah
akses phpmyadmin
pilih menu import
pilih database yang akan diimport
klik import
tunggu sampai selesai dan database siap di lihat dan di edit
